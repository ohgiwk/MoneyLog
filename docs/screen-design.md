```
│                             │
│ 最終購入日  [ 2026-05-03 ]  │
│ 次回予定日  2026年7月2日     │
│ 月額換算    ¥600             │
│                             │
│ メモ        [             ] │
│                             │
│         [保存]  [削除]      │
└─────────────────────────┘
```

**サイクルの考え方**
```
基準サイクル日数: cycle_days（例: 60日）
人数スケール有り: 実効サイクル = cycle_days ÷ household_members
次回購入予定日 = last_purchased + 実効サイクル日数
月額換算 = amount × quantity ÷ (実効サイクル日数 ÷ 30)
```

**カテゴリ選択肢（消耗品費専用）**
- 衛生・清潔（歯ブラシ、シャンプーなど）
- トイレ・洗剤（トイレットペーパー、洗剤など）
- サプリ・医療（サプリメント、常備薬など）
- 食品（調味料、米など）
- その他
